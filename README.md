# Bank-Analysis-EDA-
End to End Bank Analysis Project Using Python, PowerBI, SQL, Excel
Bank-Analysis-EDA-
End to End Bank Data Analysis (EDA) using Python, PowerBI, Excel and SQL

**ABOUT THE PROJECT**
This project analyzes a banking dataset to uncover customer behavior, lending trends, and deposit patterns using SQL, Python (EDA), and Power BI. The interactive dashboard provides insights into loan distribution, deposit analysis, and key banking KPIs, helping identify business trends and support data-driven decision-making.

The project includes data cleaning, exploratory data analysis, SQL-based querying, and interactive visualizations to transform raw banking data into meaningful business insights. It demonstrates an end-to-end data analytics workflow, from data preparation to dashboard development.

**About DataSet **
This dataset basically contains information about bank details ,various client details which consists of multiple tables which are interlinked with each other through keys like primary key and foreign key. The various tables are Banking Relationship, Client-Banking, Gender, Investment Advisor and Period.
